# llm-train-lab

`llm-train-lab` is the interactive side of the learning system.

It starts with three guided activities:

1. a shell trainer
2. a formula checker
3. a drag-and-drop transformer puzzle

## Local Dev

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

This app is designed for Vercel deployment.
